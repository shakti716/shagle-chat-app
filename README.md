# Shagle

A simple Omegle-style random chat website built with Node.js, Express, and Socket.io.

## Features

- Random anonymous pairing
- Real-time text chat
- Next chat and report user

## Run Locally

1. Install dependencies:
   ```
   npm install
   ```
2. Start the server:
   ```
   npm start
   ```
3. Open `http://localhost:3000`
